# MITRE ATT&CK Mapping

## Detection Use Case
Multiple Failed Login Attempts

## MITRE ATT&CK Tactic
Credential Access

## MITRE ATT&CK Technique
Brute Force / Password Guessing

## Purpose
Mapped the failed login detection use case to MITRE ATT&CK to understand the attacker behavior behind the alert.

## Reason
Multiple failed login attempts can indicate that an attacker is trying to guess valid credentials for a user account.

## Learning
MITRE ATT&CK helps SOC analysts classify suspicious activity based on real-world attacker tactics and techniques.
