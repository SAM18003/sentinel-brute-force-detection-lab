# Windows Endpoint Deployment

## VM Name
SOC-WIN01

## Operating System
Windows Server 2025 Datacenter

## Resource Group
RG-SOC-LAB

## Purpose
Deployed a Windows endpoint to generate security telemetry for Microsoft Sentinel monitoring and incident investigation.

## Learning
The Windows VM acts as the monitored endpoint and generates Windows Security Events, Sysmon logs, PowerShell activity, and authentication events used during threat detection and investigation.
